A flutter-based project to build critical APIs so you must know what we have in Laravel application.

## Installation (required)

It depends you must follow recommendation for your platform (Windows, macOS and Linux)

## Get dependencies then run it on your platform

```bash
flutter pub get
flutter run -d linux-or-windows-or-macos
(ex: flutter run -d windows)
```

## For development (How to run):
flutter run -d web-server --web-port=8080
